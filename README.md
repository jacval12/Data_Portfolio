## ðŸ“‚ Project Organization

All projects are organized in the `Projects` folder. Each project is contained in its **own subfolder**, and the structure may vary depending on the project type and tools used.  

Typical subfolders for each project may include:

- **`data/`** : Datasets used in the project (Excel, CSV, etc.)  
- **`code/`** : Python scripts, Jupyter notebooks, SQL queries, or other code files  
- **`dashboards/`** : Power BI dashboards (`.pbix`) or other interactive visualizations  
- **`README.md`** : Project-specific explanation, methodology, and results  

> **Note:** Not all projects will include every subfolder. For example:  
> - A Python project may have **`data/`** and **`code/`** but no **`dashboards/`**  
> - A Power BI project may have **`data/`** and **`dashboards/`** but no **`code/`**  
> - Some SQL projects may include only **`code/`** and **`data/`**  
>  
> Each project folder **always includes its own README**, explaining how to use the files, explore results, and understand the analysis.
